# Task 7 - Fetch and Display Data from Public API  
_Web Development Internship – Elevate Labs_

## Objective  
Use JavaScript **Fetch API** to get user data from a public API and display it on a webpage.

---

## project Structure

---

## API Used  
Data is fetched from the free public API:

 **https://jsonplaceholder.typicode.com/users**

---

## Features  
✔ Fetches user data using Fetch API  
✔ Displays **Name, Email, City, Street**  
✔ Includes **Reload** button to fetch data again  
✔ Handles errors (Internet off / API failure)  
✔ Clean and responsive UI using CSS  
✔ Automatic fetch on page load  

---

##  Technologies Used  
- HTML  
- CSS  
- JavaScript (Fetch API)  
- JSON  

---

##  Output Preview  
Users will be displayed in card format like:

- Name  
- Email  
- City  
- Street  

---

##  How to Run the Project  
1. Download the files or clone the repo  
2. Open the folder in **VS Code**  
3. Right-click `index.html` → **Open with Live Server**  
4. Data will appear on the page  
5. Click **Reload Users** to fetch again  

---

##  Error Handling  
If internet is turned OFF or API fails:  
- An error message will be shown  
- Error logged in console
